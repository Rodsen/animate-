# function animate
一个简单的动画函数
## 介绍
演示地址：https://rodsen.github.io/function-animate/
#### 语法
animate(element,json,time,fn)
#### 参数
- element:动画元素对象
- json: 键值对格式对象，写入动画目标属性和值
- time: （可选）动画速度，默认为20
- fn: （可选）回调函数
